  - Midbrain Academy Automation Testing Suite

1. Project Description -
This project focuses on the end-to-end automation of the Midbrain Academy web application. The goal is to ensure the stability and functionality of the platform's core features, such as user registration, course management, and navigation, using a robust automation framework.

2. Automation Tools & Technologies
Language: Java
Automation Tool: Selenium WebDriver
Framework: TestNG (For Assertion ani Reporting)
Build Tool: Maven
Design Pattern: Page Object Model (POM)
IDE: Eclipse 

3. Test Scenarios Covered
- Registration/Login: User profile creation and login functionality.
- Course Search: search for Specific courses  and to verify the results.
- Enrollment Flow: check the enrollment process after courses section.
- Contact Us Form: Form fill-up and submission validation.
- UI/Navigation: Check the all links and menu bar correctly navigate or not.
- Responsiv: Browser and Mobile

4. Project Folders
Pages: Contains the location of web elements (Buttons, Text boxes).
Tests: Contains the actual test scripts.
Resources: Contains the website URL and browser settings.

5. How to Run the Project
- Download/Clone the project from GitHub.
- Open it in Eclipse or IntelliJ.
- Right-click on the testng.xml file.
- Select Run As > TestNG Suite.
